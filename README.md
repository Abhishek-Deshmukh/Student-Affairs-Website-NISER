# student_affairs

## Project setup
```
$ yarrn install
$ gatsby develop
```
The site can then be accessed at localhost:8000/ .

## Update procedure

Update the word document and convert it into markdown using Add-on.

Paste the output into `./src/templates/input.md`

```
$ npm run feed
```
